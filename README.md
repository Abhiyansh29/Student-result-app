# Student Result Management App

A simple React application to manage student results using JSON Server.

## Features
- View all students
- Add new student
- Edit existing student
- Delete student
- View student details

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the JSON Server (Backend):
   ```bash
   npm run server
   ```
   This runs on `http://localhost:3000`.

3. Start the React App (Frontend):
   ```bash
   npm run dev
   ```
   This runs on `http://localhost:5173`.

## Usage
- Click "Load Students" to fetch the data.
- Use the buttons to Add, Edit, Delete, or View students.
